# React Adv

React 19 + Vite 8 + React Router 7 + TypeScript + Vitest 4

## Scripts

### `npm run dev`

Levanta la app en modo desarrollo.
Abre [http://localhost:5173](http://localhost:5173) en el navegador.

### `npm run build`

Compila TypeScript y genera el bundle de producción en la carpeta `dist/`.

### `npm run preview`

Sirve el build de producción localmente para revisarlo antes de desplegar.

### `npm test`

Lanza Vitest en modo watch interactivo.

### `npm run coverage`

Ejecuta los tests una vez y genera reporte de cobertura.
